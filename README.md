# YouTube-Opti-Toolkit

This repository contains two projects aimed at enhancing the YouTube experience.

## 1. YouTube Extension for Overflow

### Overview

The YouTube Extension for Overflow is a browser extension designed to enhance your YouTube experience by providing additional features and improvements.

## 2. YouTube Screenshot

### Overview

YouTube Screenshot is a tool that allows users to capture screenshots from YouTube videos.
